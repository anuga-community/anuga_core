Visualisation
=============

ANUGA provides several tools for visualising simulation results. During a simulation run, 
the `domain` object can generate plots of the current state of the simulation.
After the simulation, the results can be visualised using various tools, including:
the `anuga_viewer`, via a python interface using libraries such as 'Sww_Plotter', and QGIS.
