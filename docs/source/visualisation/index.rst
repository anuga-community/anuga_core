Visualisation
=============

ANUGA provides several tools for visualising simulation results. During a simulation run, 
the `domain` object can generate plots of the current state of the simulation.
After the simulation, the results of the `sww` files can be visualised a number of ways, including:
a python interface using the module  `anuga.SWW_Plotter`, the `anuga_viewer`, and QGIS.
